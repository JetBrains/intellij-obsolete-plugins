Hardcoded Struts Action URL. Reports patterns, where Struts Action URL is made "by hand" and suggests to convert using Struts <url>-tag.

Example:


<a href="namespace/foo!bar.action?param1=value1&param2=value2"/>

After the quick-fix is applied:


<a href="<s:url namespace="namespace" action="foo" method="bar"/>
  <s:param name="param1">value1</s:param>
  <s:param name="param2">value2</s:param>
</s:url>"/>
