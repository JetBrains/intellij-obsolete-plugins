Reports custom tags that do not exist.

You can create custom tags for your application using simple template files stored in the app/views/tags directory. The template's file name is used as the tag name. For example, to create a hello tag, just create the app/views/tags/hello.html file. Reference the tag directly: #{hello /}.

This inspection will report the #{hello /} reference if app/views/tags/hello.html does not exist.
