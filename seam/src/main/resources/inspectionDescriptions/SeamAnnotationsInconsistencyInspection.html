Reports Seam annotation usages on invalid component types.

Seam annotations are allowed only for classes that are Seam components (except @Around, @Within, @DataBinderClass, and @DataSelectorClass).
Additionally, many annotations are only valid for specific component types, for example, Entity Beans do not support bijection, context demarcation, or transaction demarcation.

The following component types are allowed for Seam annotations:
