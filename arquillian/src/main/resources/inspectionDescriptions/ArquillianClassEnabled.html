Reports test classes that are not prepared for testing with Arquillian.

According to the Arquillian documentation, test classes should either be annotated as @RunWith(Arquillian.class) (for a JUnit version of Arquillian) or inherited from the org.jboss.arquillian.testng.Arquillian class (for a TestNG version of Arquillian).

The quick-fix either declares the test class abstract or makes it comply with the requirements.
